## Installation

### Clone the Repository
```
git clone git@github.com:N1lsE/.dotfiles.git ~/.dotfiles
```

### create symlinks
Make shure every symlink you want to create via the `make` command (makefile) doesn't exist befor running.
```
cd ~/dotfiles
make
cd ~
```
